# 3D Visualization Dashboard
![Uploading 截屏2025-08-05 16.12.33.png…]()

这是一个基于React和Three.js的3D可视化仪表盘项目。

## 技术栈
- React 18
- React Router
- Three.js (3D可视化)
- Recharts (图表展示)
- React Scripts (构建工具)

## 安装步骤
1. 克隆仓库
```bash
https://github.com/yarenxinqing/3D-Visualization-Dashboard.git
cd 3D-Visualization-Dashboard
```

2. 安装依赖
```bash
npm install
```

## 运行项目
```bash
npm start
```
项目将在 http://localhost:3001 启动

## 构建项目
```bash
npm run build
```
构建后的文件将保存在 `build` 目录中

## 项目结构
- `public/`: 静态资源
- `src/`: 源代码
  - `components/`: 可复用组件
  - `pages/`: 页面组件
  - `App.js`: 应用入口组件
  - `index.js`: 渲染入口
